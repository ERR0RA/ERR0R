Algebra

Let's start off with simple equations. An example of a simple equation is x + 3 = 8.

To work this out we need to bring 3 to the other side, so we subtract 3 from both sides to get x + 3 - 3 = 8 - 3.

This makes x = 5.

Here is another example (the answer will be told later in the harder questions): x + 3 = 8.

Now let's look at another equation, but with a negative sign.

x - 5 = 9

To remove the 5 we need to add 5 to the x side, and add 5 to the other side.

So we get x - 5 + 5 = 9 + 5.

This gets us to x = 14.

Answer for example questions before this one is x = 5
